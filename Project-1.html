O básico da internet

A web é uma coleção de documentos do tipo HTML. Por isso, documentos do tipo HTML podem ser chamados do coração da Web. A sigla HTML siginifica Hypertext Markup Language HTML é a linguagem usada pelos navegadores para interpretar a web.
Os componentes básicos da web são: clientes (eu e meu browser), internet e servidores. A comunicação entre meu navegador e os servidores é feita através da internet que usa HTTP (Hypertext Transfer Protocol) para fazer a comunicação entre meu browser e os servidores

Elementos de um documento HTML

Elementos de HTML são um tipo de estrutura, na verdade é a estrutura que é usada para dizer para o navegador como a página deve ser mostrada. Essa estrutura é basicamente composta por uma tag de abertura o conteúdo e uma tag de fechamento
Algumas tags aprendidas no curso:
tag <b> - transforma o texto em negrito
tag <em> - transforma o texto em itálico
tag <br> - separa texto em
diferentes linhas (como usado aqui)
tag <p> - separa o texto em parágrafos

Atríbutos

Atributos são valores que podemos atribuir para alguns tipos de elementos. A tag <a>, usada para link, é um exemplo de tag que pode receber um atributo. O atributo que ela recebe é href que vai nos dar o endereço do link. Outra tag que pode receber atríbutos é a tag <img>, mostra uma imagem hospedada em um site ou mesmo em uma pasta de computador. O atríbuto dessa tag é src, e vai fornecer onde está o arquivo de imagem (página da web ou pasta do computador). Para evitar problemas no caso da imagem não aparecer, depois do endereço da imagem deve ser colocacod outor atríbuto chamado alt, que vai permitir mostar um texto atribuído a imagem.
